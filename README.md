# **EMU[lator] SURGE**

This repo is planned to be a multi-console emulator that will properly run ROMs from a large number of systems.

## Planned systems currently include:
- NES
- SNES
- GBA
- NDS
- N64
- PS1
- PS2

At the moment, ***NONE OF THESE ARE WORKING***. NES Emulation is currently in progress, and I intend to follow
immediately with SNES or GBA (I haven't decided yet).

## Planned features:
- central UI to select whatever rom you intend to run (actual design is pending)
- debugging tools to aid in homebrew rom creation for any implemented systems

## Problems/Questions/Comments
If you have ***any*** issues and/or suggestions, please feel free to submit an issue detailing everything you possibly can.
